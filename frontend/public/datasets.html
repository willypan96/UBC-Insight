Manage Datasets

Add Dataset

Here is where you are able to add a dataset to the system.

1. Set Dataset ID

Here is where you specify the ID of the dataset you would like to add. A valid dataset ID must satisfy the following conditions:
  1. The ID must be non-empty.
  2. The ID must be not entirely whitespace.
  3. The ID must not contain an underscore ("_").

2. Set Dataset Kind

Next, choose the dataset kind of your dataset. The dataset kind must be one of the following:
  1. The kind must be "courses".
  2. The kind must be "rooms".

3. Select Valid Dataset Zip

The dataset must be a valid .zip file, and should contain data corresponding to the dataset kind selected above.

4. Now upload your dataset!



If everything went to plan, you should receive an alert stating that your dataset has been added successfully! Otherwise, an alert will state what went wrong.

Remove Dataset

This will remove the dataset with the specified ID from the system.

1. Set Dataset ID

Here is where you specify the ID of the dataset you would like to delete. A valid dataset ID must satisfy the following conditions:
  1. The ID must be non-empty.
  2. The ID must be not entirely whitespace.
  3. The ID must not contain an underscore ("_").
  4. A dataset with the given ID must exist in the system.

2. Remove Dataset



If everything went to plan, you should receive an alert stating that your dataset has been removed successfully! Otherwise, an alert will state what went wrong.
